WeatherPy Summary

WeatherPy was a project that used API calls, the Python pandas library, and matplotlib to show the relationship between temperature and distance from the equator.

I used Python to randomly generate 500 latitude and longitude points. Then I used the CityPy library to generate city names associated with the lat/long points. Finally, the API call to Open Weather Map library generated weather information, including temperature.

Using matplotlib, I created several visuals to display the relationship between latitude in relation to temperature, humidity, wind speed, and cloudiness.
